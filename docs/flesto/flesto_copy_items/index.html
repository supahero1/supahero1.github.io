Synopsis

#include <shnet/flesto.h>

void
flesto_copy_items(struct flesto* flesto, unsigned long dest_part,
                  unsigned long dest_offset, unsigned long src_part,
                  unsigned long src_offset, unsigned long amount_of_items,
                  unsigned long item_size);

Description

The flesto_copy_items() function shall copy amount_of_items items, each of size item_size, from src_part and src_offset to dest_part and dest_offset.

Due to how flesto operates, memcpy() cannot be used consistently if the amount_of_items exceeds 1.

This function will skip any insufficient free space at the end of any part encountered on its copying path, just like flesto_add_item().

flesto_copy_items() will not check if dest_part + dest_offset + amount_of_items * item_size (simplification) exceeds the last item of the flesto. If it does, the behavior is undefined.

This function cannot be used to copy data between different flestos.

Return value

flesto_copy_items() does not return anything.

Errors

flesto_copy_items() always succeeds.
