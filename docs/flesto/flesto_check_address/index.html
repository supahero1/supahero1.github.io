Synopsis

#include <shnet/flesto.h>

int flesto_check_address(struct flesto* flesto, void* ptr);

Description

The flesto_check_address() function shall make sure the ptr belongs to the used space of the flesto, that is, belongs to parts from the first to the last (do not confuse the last (used) part with the last part of the array of parts) and does not exceed the capacity of the part it belongs to.

This function is the least permissive amongst the 4. It allows the ptr to only be in the used space of the flesto.

Return value

flesto_check_address() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_failure - the ptr does not meet the requirements
