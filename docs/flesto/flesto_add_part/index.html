Synopsis

#include <shnet/flesto.h>

int flesto_add_part(struct flesto* flesto, char* contents,
                    unsigned long max_capacity);

Description

The flesto_add_part() function shall grow given flesto's array of parts by 1 and, at the end of the array, create a new part using the contents and max_capacity. The new part shall have no data. Its capacity is set to 0.

This function is similar to flesto_resize(), but flesto_add_part() allows for any size of the new part.

Use flesto_add_parts() if you need to add multiple parts.

Return value

flesto_add_part() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_out_of_memory - an out of memory error has occured while expanding the array of parts

Examples

struct flesto f = flesto(5, 0);

void* content = malloc(100);
if(content == NULL) {
  exit(1);
}

int err = flesto_add_part(&f, content, 100);
if(err != flesto_success) {
  exit(1);
}
