Synopsis

#include <shnet/flesto.h>

int flesto_resize(struct flesto* flesto, unsigned long new_size);

Description

The flesto_resize() function shall resize given flesto's array of parts to match the specified size new_size.

If the array is grown, the new parts will have the size of flesto->default_max_capacity and will be empty. Last part's index flesto->last_part will remain unchanged.

If the array is shrunk, parts not fitting the new size will have their contents freed and flesto->last_part, if affected by the shrinking, will point to the last part in the flesto.

new_size must be greater than 0, otherwise it results in undefined behavior (use flesto_free() instead).

Return value

flesto_resize() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_out_of_memory - an out of memory error has occured
