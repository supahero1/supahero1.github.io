Synopsis

#include <shnet/flesto.h>

int flesto_delete_after(struct flesto* flesto);

Description

The flesto_delete_after() function shall determine if there is an unused part after the last part of the flesto. If there is, flesto_delete_after() shall then calculate how much free space the last part has (last_part.max_capacity - last_part.capacity). If the amount is greater or equal to the flesto->delete_after property, the unused part after the last part is removed from the flesto and the array of parts is shrunk by 1.

This might be helpful in situation like described in the trivia section of the flesto documentation. By increasing the window of potential data changes, parts will not be potentially freed and allocated constantly, which could otherwise become a bottleneck.

Benchmark your application before picking a value for flesto->delete_after for maximum efficiency.

Return value

flesto_delete_after() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_out_of_memory - an out of memory error has occured while shrinking the array of parts
