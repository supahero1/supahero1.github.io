Synopsis

#include <shnet/flesto.h>

void flesto_pop(struct flesto* flesto, unsigned long amount);

Description

The flesto_pop() function shall adjust the flesto so that amount bytes, counting from the end, are deleted.

If you need to delete data from elsewhere than the end of the flesto, see flesto_remove_items().

This function only adjusts capacities of parts and the flesto->last_part property. It will not remove empty parts. If you need to do so, see flesto_remove_empty_parts().

This function shall not fail if amount is greater than the total amount of bytes the flesto houses - it will stop when flesto->last_part is detected to be 0 and there are more bytes to delete.

Return value

flesto_pop() does not return anything.

Errors

flesto_pop() always succeeds.
