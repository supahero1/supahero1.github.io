Synopsis

#include <shnet/flesto.h>

void flesto_remove_items(struct flesto* flesto, void* items,
                         unsigned long part, unsigned long offset,
                         unsigned long amount_of_items,
                         unsigned long item_size);

Description

See flesto_remove_item() if you want to remove only one item, or for reference. flesto_remove_items() matches the entire description of flesto_remove_item() if not stated otherwise below.

The flesto_remove_items() function shall remove the items under specified address items or under specified part and offset from the given flesto. That might happen in 3 ways:

1. items + amount_of_items * item_size is greater or equal to the last item of the flesto, meaning there are no items to copy:

before:
==========================================
|                      | items_to_delete |
==========================================
after:
========================
|                      |
========================

2. items + amount_of_items * item_size is less than the last item of the flesto, but items + amount_of_items * item_size * 2 is greater than the last item of the flesto, meaning the gap can be filled only partially:

before (cp = items_to_copy):
==========================================
|                 | items_to_delete | cp |
==========================================
after:
========================
|                 | cp |
========================

3. items + amount_of_items * item_size is less than the last item of the flesto, and items + amount_of_items * item_size * 2 is less or equal to the last item of the flesto, meaning the gap can be fully filled:

before:
==========================================
| items_to_delete |    |  items_to_copy  |
==========================================
after:
========================
|  items_to_copy  |    |
========================

Error checking for the item is performed only partially. The function will detect if the pointer does not belong to the flesto, but it will not detect if the pointer belongs to the unused space. Use flesto_check_address() to make sure the pointer belongs to the flesto's used space.

items can be NULL, but then the user must provide valid part and offset arguments, denoting part and offset (index) from the beginning of the part (not the beginning of all the items in the flesto) respectively. These values can be received from the flesto_ptr_to_idx() function. If items is not NULL, the other 2 arguments are ignored.

Return value

flesto_remove_items() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_out_of_memory - an out of memory error has occured

If items is not NULL:

flesto_failure - items does not belong to any of the flesto's parts
