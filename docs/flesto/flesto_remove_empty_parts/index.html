Synopsis

#include <shnet/flesto.h>

int flesto_remove_empty_parts(struct flesto* flesto);

Description

The flesto_remove_empty_parts() function shall free the contents of every part between flesto->max_amount_of_parts and flesto->last_part, even if the parts contain any data.

Return value

flesto_remove_empty_parts() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_out_of_memory - an out of memory error has occured while shrinking the array of parts
