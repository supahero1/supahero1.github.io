Synopsis

#include <shnet/flesto.h>

int flesto_ptr_to_idx(struct flesto* flesto, void* ptr,
                      unsigned long* part, unsigned long* offset,
                      unsigned long tip, int dir);

Description

The flesto_ptr_to_idx() function shall find the part and offset of the given ptr.

If it is known what part might be near the ptr, the tip and dir arguments can be used, where tip is a part index and dir allows the user to specify whether the ptr is to the left or the right of the tip (flesto_left and flesto_right respectively). If unused, tip and dir must be 0 and flesto_right respectively.

Return value

flesto_ptr_to_idx() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_failure - the ptr does not belong to the used space of the flesto

Notes

See flesto_idx_to_ptr() for more information.
