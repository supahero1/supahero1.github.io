Synopsis

#include <shnet/flesto.h>

void flesto_free(struct flesto* flesto);

Description

The flesto_free() function shall free the contents of any parts the flesto has, then free the array of parts, and lastly reset flesto->parts, flesto->max_capacity, and flesto->capacity, by setting them to NULL and zeroes respectively.

Return value

flesto_free() does not return anything.

Errors

flesto_free() always succeeds.
