#include <shnet/flesto.h>

int flesto_check_address2(struct flesto* flesto, void* ptr);

Description

The flesto_check_address2() function is similar to flesto_check_address(), but this function does not allow the ptr to exceed the last part of the array of parts instead of the last used part.

Return value

flesto_check_address2() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_failure - the ptr does not meet the requirements
