Synopsis

#include <shnet/flesto.h>

unsigned long flesto_amount_of_items(struct flesto* flesto);

Description

The flesto_amount_of_items() function shall add the capacities of every part in the flesto (up to the last part) and then return the result.

Return value

flesto_amount_of_items() returns the amount of bytes used in the flesto.

Errors

flesto_amount_of_items() always succeeds.
