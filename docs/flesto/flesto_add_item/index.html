Synopsis

#include <shnet/flesto.h>

int flesto_add_item(struct flesto* flesto, const void* item,
                    unsigned long item_size);

Description

The flesto_add_item() function shall copy item_size bytes from item to the end of flesto's last part. If necessary, it will add a new part using flesto_resize()

If the last part has free space, but which is less than the requested item_size, it will skip that free space. If the user knows this can occur, the part's contents may be resized to fit its capacity in order to preserve memory after this function returns.

If there is an empty part after the last part, and the last part does not satisfy the memory needs, the empty part will be used and marked as the new last part.

item_size must be smaller than a part's size, else it results in undefined behavior.

To add multiple items at the same time, use flesto_add_items().

Return value

flesto_add_item() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_out_of_memory - an out of memory error has occured
