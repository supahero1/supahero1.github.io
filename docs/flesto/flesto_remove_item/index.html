Synopsis

#include <shnet/flesto.h>

void flesto_remove_item(struct flesto* flesto, void* item,
                        unsigned long item_size);

Description

The flesto_remove_item() function shall remove the item under specified address item from the given flesto. That might happen in 2 ways:

1. item is the last item of the flesto, nothing happens

2. item is not the last item of the flesto, so the last item is copied to item using memcpy()

The last part is then adjusted by item_size.

If the last part's capacity ends up being 0, and the last part is not the only part, flesto->last_part is decreased by 1. The old last part is not freed.

Copying is used to disallow gaps in flesto. With time, such gaps could potentially grow to large regions of unused memory. This also means all the items must be of the same size. If they are not, and the item is not the last item of the flesto, it results in undefined behavior.

item_size must be smaller than a part's size, else it results in undefined behavior.

If you have the item's index instead of its address, use flesto_idx_to_ptr() to convert it to a pointer.

Error checking for the item is not performed. If it is not a valid address or does not belong to the flesto, the resulting behavior is undefined. Use flesto_check_address() to make sure the pointer belongs to the flesto.

To remove multiple items at the same time, see flesto_remove_items().

Return value

flesto_remove_item() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_out_of_memory - an out of memory error has occured
