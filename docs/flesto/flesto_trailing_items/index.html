Synopsis

#include <shnet/flesto.h>

unsigned long
flesto_trailing_items(struct flesto* flesto, unsigned long part,
                      unsigned long offset);

Description

The flesto_trailing_items() function shall calculate the amount of bytes until the last item of the flesto, counting from the given part and offset from the beginning of the part.

If you have a pointer instead of part and offset indexes, see flesto_ptr_to_idx().

Both part and offset must be valid, otherwise it results in undefined behavior.

Return value

flesto_trailing_items() returns the amount of bytes from the specified point to the end of the flesto.

Errors

flesto_trailing_items() always succeeds.
