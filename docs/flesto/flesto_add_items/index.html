Synopsis

#include <shnet/flesto.h>

int flesto_add_items(struct flesto* flesto, const void* items,
                     unsigned long amount_of_items,
                     unsigned long item_size);

Description

See flesto_add_item().

The flesto_add_items() function shall copy amount_of_items * item_size bytes from items to the end of flesto's last part. If necessary, it will add one or more parts using flesto_resize()

If the last part has free space, but which is less than the requested item_size, it will skip that free space. If the user knows this can occur, the part's contents may be resized to fit its capacity in order to preserve memory after this function returns. However, it is up to the user to find one or more of such parts, because after this function returns, the amount of parts might increase by more than 1.

If there is an empty part after the last part, and the last part does not satisfy the memory needs, the empty part will be used and marked as the new last part.

item_size must be smaller than a part's size, else it results in undefined behavior.

To add only one item, use flesto_add_item().

Return value

flesto_add_items() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_out_of_memory - an out of memory error has occured
