Synopsis

#include <shnet/flesto.h>

struct flesto flesto(unsigned long default_max_capacity,
                     unsigned long delete_after);

Description

The flesto() function shall return a struct flesto prefilled with default values and those given by the user:

struct flesto {
  struct flesto_part* parts = NULL,
  unsigned long last_part = 0,
  unsigned long max_amount_of_parts = 0,
  unsigned long default_max_capacity = default_max_capacity,
  unsigned long delete_after = delete_after
};

The user may omit this function and create the struct by themselves.

Return value

This function returns a filled struct flesto.
