Synopsis

#include <shnet/flesto.h>

int flesto_check_address3(struct flesto* flesto, void* ptr);

Description

The flesto_check_address3() function is similar to flesto_check_address2(), but this function does not allow the ptr to exceed the maximum capacity instead of the capacity of the part it belongs to.

This function is the most permissive amongst the 4. It allows the ptr to be anywhere in the flesto.

Return value

flesto_check_address3() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_failure - the ptr does not meet the requirements
