Synopsis

#include <shnet/flesto.h>

void*
flesto_idx_to_ptr(struct flesto* flesto, unsigned long ref_part,
                  unsigned long ref_offset, unsigned long idx,
                  int dir, unsigned long* part, unsigned long* offset);

Description

The flesto_idx_to_ptr() function shall calculate what place ref_part + ref_offset + idx (simplification) points to.

Since flesto consists of parts instead of continuous memory, one cannot simply add the idx to the ref_offset. This function helps with that.

If part is not NULL, the part the calculated place belongs to will be stored at the location part points to. If it is not a valid pointer and is not NULL, the behavior is undefined.

If offset is not NULL, the offset of the calculated place, counting from the beginning of the part, will be stored at the location offset points to. If it is not a valid pointer and is not NULL, the behavior is undefined.

dir allows the user to specify in what direction the function should move. It can be either flesto_right or flesto_left.

Return value

flesto_idx_to_ptr() returns a pointer to the resulting place in the flesto, or NULL in case of an error.

Errors

NULL - the specified indexes and dir, in combination with specified ref_part and ref_offset, are not within the boundaries of the flesto
