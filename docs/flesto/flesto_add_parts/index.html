Synopsis

#include <shnet/flesto.h>

int flesto_add_parts(struct flesto* flesto, char** contents,
                    unsigned long* max_capacity, unsigned long amount);

Description

The flesto_add_parts() function shall grow given flesto's array of parts by amount and, at the end of the array, create amount new parts using the contents and max_capacity. The new parts shall have no data. Their capacities are set to 0.

This function is similar to flesto_resize(), but flesto_add_parts() allows for any sizes of the new parts.

Use flesto_add_part() if you need to add only one part.

Return value

flesto_add_parts() returns flesto_success on success, otherwise one of the error codes.

Errors

flesto_out_of_memory - an out of memory error has occured while expanding the array of parts

Examples

struct flesto f = flesto(5, 0);

void* contents[3];
contents[0] = malloc(10);
contents[1] = malloc(23);
contents[2] = malloc(123);
if(contents[0] == NULL || contents[1] == NULL
                       || contents[2] == NULL) {
  exit(1);
}

unsigned long sizes[] = { 10, 23, 123 };

int err = flesto_add_parts(&f, contents, sizes, 3);
if(err != flesto_success) {
  exit(1);
}
