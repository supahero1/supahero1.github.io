Synopsis

#include <shnet/misc.h>

void mufex_unlock(struct mufex* mx, int shared);

Description

The mufex_unlock() function shall unlock the mx based on the value of shared.

If shared is mufex_shared, internal counter of the mx will be decreased, but if it is not 0 (i.e. this call is not the last call that was holding the lock), any calls made using mufex_not_shared will continue to block.

If shared is mufex_not_shared, the lock is released and any other call can instantly aquire it (just like a normal mutex).

Return value

mufex_unlock() does not return anything. It always succeeds.
