Synopsis

#include <shnet/misc.h>

int contmem_pop(struct contmem* contmem, void* mem);

Description

The contmem_pop() function shall decrease the number of items in the contmem by 1, removing the last block if necessary. If mem is not equal to the last used item in the contmem, the function will also copy the last item to mem to fill the gap. Applications using this interface need to be prepared for that and, if they are using pointers, change them appropriately.

Return value

contmem_pop() returns 0 if mem is the last item of the contmem and thus its location did not change, otherwise 1. It always succeeds.
