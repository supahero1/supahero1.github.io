Synopsis

#include <shnet/misc.h>

void* contmem_get(struct contmem* contmem);

Description

The contmem_get() function shall allocate a new block if necessary and then return a pointer to the first available free space in the structure. The free space will always be at the end (see contmem_pop()) and will have the size of contmem->item_size. Using more than that results in undefined behavior.

Return value

contmem_get() returns pointer to the requested memory, otherwise one of the return codes.

Errors

NULL - an out of memory error has occured
