Synopsis

#include <shnet/misc.h>

int contmem(struct contmem* contmem, unsigned long max_items,
            unsigned long item_size, unsigned long tolerance);

Description

The contmem() function shall create a new struct contmem, initialise its members, and lastly store it at contmem. max_items specifies maximum size of a block, item_size specifies size of any object that will be stored in the structure. Memory requested by contmem_get() will only be item_size bytes wide - using more than that results in undefined behavior.

tolerance specifies how many bytes need to become unused in order to free the last unused block, if there is any. If it is 0, any unused block will be disgarded as soon as it becomes empty. If it is larger than 0, the structure will wait until tolerance bytes become free in the last used block. If it comes true, it will then check if there is any unused block after the last used one. If yes, it will free and remove it.

contmem() shall also allocate one block of memory.

Return value

contmem() returns contmem_success on success, otherwise one of the error codes.

Errors

contmem_out_of_memory - an out of memory error has occured
