Synopsis

#include <shnet/misc.h>

void* contmem_last(struct contmem* contmem);

Description

The contmem_last() function shall return pointer to the first unused (the end of the last used) item in the contmem. Never use it on your own - use contmem_get() instead.

Return value

contmem_last() returns pointer to the first unused item in the contmem. It always succeeds.
