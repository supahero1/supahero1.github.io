Synopsis

#include <shnet/misc.h>

void mufex_lock(struct mufex* mx, int shared);

Description

The mufex_lock() function shall lock the mx based on the value of shared.

If shared is mufex_shared, all further attempts to lock the mx using mufex_shared too will succeed, including the current one. If the mx is already locked using mufex_not_shared, mufex_lock() will block until the mx is released.

If shared is mufex_not_shared, all further attempts to lock the mx using any shared value will fail (true mutual exclusion). If the mx is already locked using mufex_shared, mufex_lock() will block until the mx is released.

Return value

mufex_lock() does not return anything. It always succeeds.
